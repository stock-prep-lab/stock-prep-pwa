# 要件概要

## アプリの目的

日本株を主対象に、引け後や土日に分析し、翌営業日の朝に仕込む候補を決める。
日足データが取得できる範囲で、米国株、英国株、香港株も同じ分析導線に広げられるようにする。

## 目的詳細

- 条件に合う候補銘柄を見つける
- 候補銘柄のトレンドを確認する
- 現在の保有ポートフォリオを把握する
- リバランス候補を提案する
- 購入した場合の構成変化をシミュレーションする
- 通知で毎日の確認を助ける

## 想定利用シーン

- 夜にアプリを開く
- 今日の候補銘柄を確認する
- 気になる銘柄の詳細チャートを見る
- 自分の保有ポートフォリオを確認する
- リバランス候補を確認する
- 購入価格と株数を入力して購入後の構成変化を確認する
- 翌営業日の朝に仕込む判断をする

## 対象外

- リアルタイムの場中売買支援
- 自動売買
- 高度なファンダメンタル分析
- ニュース分析
- 複数ユーザー向け SaaS 化
- プロ向け高速トレードツール化

## データ前提

- 日次データ中心
- 価格系列ベースで分析する
- 日本株を主対象にしつつ、米国株、英国株、香港株の日足も取り込み対象に含める
- 外貨建て保有のポートフォリオ評価には為替日次を利用する
- 外部データソースの第一候補は Stooq daily ASCII bulk とする
- Stooq bulk は管理者が手動取得してアプリへアップロードする
- 全銘柄完全保証は前提にしない
- 取得できる範囲のデータを活用する

## 提供形態

- PWA
- スマホ中心利用
- デスクトップでも利用可能
- 個人用
- デプロイは Vercel
